🚍 BAUST Trip Connect
📌 Overview

BAUST Trip Connect is a web-based internal transportation system developed for the Bangladesh Army University of Science and Technology (BAUST). The platform enables real-time bus tracking, route management, and schedule updates to improve campus travel efficiency for students and staff. Built with Vue.js (Frontend), Spring Boot (Backend), and PostgreSQL (Database), the system ensures reliability, scalability, and a user-friendly experience.

✨ Features
Real-time bus location tracking using mobile latitude/longitude updates
Route details and schedule information for passengers
Notifications to reduce waiting time and uncertainty
Role-based access for administrators, staff, and passengers
Centralized admin panel for bus, route, and schedule management
Online payment integration for seamless commuting
🛠️ Tech Stack
Technology	Usage
Vue.js	Frontend
Spring Boot	Backend
PostgreSQL	Database
⚙️ How It Works
👨‍💼 Administrators

Administrators manage buses, assign routes, update schedules, and oversee user data.

👨‍🔧 Staff

Staff provide live location updates, processed by the backend at regular intervals.

🎓 Passengers

Passengers use a responsive interface to track buses, view routes, receive notifications, and plan journeys efficiently.

🎯 Objectives
Provide real-time bus tracking for BAUST community
Implement secure role-based access control
Automate transportation workflows to reduce manual effort
Enhance campus productivity and convenience
🚀 Future Scope
Advanced analytics for bus usage and route optimization
Enhanced mobile app integration
Improved notification system with predictive arrival times
📖 Summary

BAUST Trip Connect was designed to solve campus transportation challenges by integrating live tracking, secure access, and centralized management. It improves coordination among stakeholders, reduces uncertainty, and delivers a modern solution for campus mobility.
